# üçÖ Pomodoro Timer Web App

This is a Pomodoro Timer Web App built with vanilla JavaScript, HTML, and CSS.
It helps users manage work sessions and breaks efficiently using the Pomodoro technique.
The timer is interactive, fully functional, and includes automatic session switching, reset functionality, and multiple session modes.

## ‚ú® Features
- Pomodoro Mode: 50-minute focused work sessions.
- Short Break Mode: 5-minute short breaks
- Long Break Mode: 10-minute long breaks
- Single Start Button: Starts the currently selected session
- Automatic Session Switching: After a Pomodoro finishes, a short break starts automatically, and vice versa.
- Reset Button: Resets the timer to the current session without changing the session type.
- Mode Switching: Users can switch between Pomodoro, short break, and long break without starting the timer automatically.
